## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR-USERNAME/calcnova.git

# Open in browser (no build step needed)
cd calcnova
open index.html
```

> This project runs entirely in the browser — no Node.js, no backend, no dependencies.
